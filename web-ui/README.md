```

### Menu

+-----------------------------+
```

### MatchWaitingPage
```
+-----------------------------+
|    正在为你匹配对手...        |
|         [ 取消匹配 ]         |
+-----------------------------+
```

### QuizRoomPage
```
+----------------------------------+
|你：xxx (2分) VS  对手：yyy (1分)  |
|----------------------------------|
| 问题类型 1/20：                   |
|   问题内容                        |
|           [ 提交 ] [ 放弃 ]       |
+----------------------------------+
```

规则：
先答对的人多得分（+2）；
答对得3分，答错扣一分；
跳过不得分。

### ResultPage
```
+-----------------------------+
|         胜利 / 失败          | 
|      *  xxx 10分             |
|          yyy  9分            |
|    [ 继续PK ] [ 返回首页 ]    |
+-----------------------------+
```
